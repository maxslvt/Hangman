# Hangman

This is a simple bash version of the game "Hangman".

## How to install

### Copy/Paste

```bash
gcl https://github.com/maxslvt/Hangman
```

## ENJOY!

Create the executable with :

```bash
make
```

Execute like this (lists are stored in ./assets/ and you can easily create more):

```bash
./hangman [list]
```
